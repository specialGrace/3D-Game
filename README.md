# CSS 3D Maze Explorer

A fun 3D game built with HTML, pure vanilla JavaScript and CSS 3D transforms (no game engines or heavy libraries).

## Features
- Smooth first-person movement and mouse look (pointer lock)
- Collision detection with maze walls
- Collect coins, keys, dogs, and cartoon items
- Rotating collectibles
- Atmospheric glowing wall decorations with runes
- Sound effects for collecting items
- Reset functionality (`R` key)

## How to Play

- **Arrow Keys** — Move around
- **Mouse** — Look around (click to lock cursor)
- **Space** — Jump / Go up
- **P** — Speed boost
- **R** — Reset position

Goal: Explore the labyrinth and collect all the items!

## Controls

| Key          | Action              |
|--------------|---------------------|
| Arrow Keys   | Move                |
| Mouse        | Look around         |
| Space        | Jump                |
| P            | Speed boost         |
| R            | Reset position      |
| Click        | Lock mouse cursor   |

## Tech Stack

- HTML5
- CSS 3D Transforms
- Vanilla JavaScript
- Web Audio API (for sounds)

Made as a creative experiment with CSS 3D.
